1. DASHBOARD & FITUR OPERASIONAL
═══════════════════════════════════════════════════════════════
Dashboard aktif per role:
  Staff, Head Store, District Manager, Area Manager,
  Ops Manager, Finance, Supply Chain, Trainer, Auditor.

Fitur utama:
- Ceklis harian 3 shift (pagi/middle/malam) + foto per item

- Visit monitoring DM/Area: daily visit, audit toko (scoring 110 poin)

- Store Status real-time di /dm/stores
- Pengumuman: input di /ops/pengumuman, tampil di semua staff dashboard
- Setoran detail /ops/setoran: date selector, search, filter chips
  Status mapping daily_deposits:
    approved + selisih=0  → Selesai (hijau)
    approved + selisih≠0  → Selisih (merah)
    submitted             → Pending (oranye)
    rejected              → Ditolak (rose)
    no record / draft     → Belum (abu)
- Data Explorer di /ops: 8 tabel analitik untuk ops_manager
